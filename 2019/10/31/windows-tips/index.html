发表于 2019-10-31 分类于 windows 阅读次数：
windows问题：PowerShell yarn : 无法加载文件 C:\Users\Admin\AppData\Roaming\npm\yarn.ps1,因为在此系统因为在此系统上禁止运行脚本的问题

今天在windows系统上执行yarn之类的命令时突然 报无法加载文件，因为在此系统因为在此系统上禁止运行脚本 错误。本着跟错误死磕到底的精神，科学上网了一波，解决了问题，特此记录一下。

1.打开powershell，以管理员身份运行

2.若要在本地计算机上运行您编写的未签名脚本和来自其他用户的签名脚本，请使用以下命令将计算机上的 执行策略更改为 RemoteSigned
执行：set-ExecutionPolicy RemoteSigned
输入 y

3、查看执行策略：get-ExecutionPolicy

4.关闭命令窗口 即可

然后就可以愉快的使用命令啦！
